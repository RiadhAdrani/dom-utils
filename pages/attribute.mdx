# Attributes

Handle element's attributes.

---

### `convertAttributeToDomProperty()`

convert attribute to dom property.

if the attribute is not found, we return the camel-cased version of the attribute.

```ts
function (attr: string): string

```

- `attr` : html attribute.

### `isDataAttr()`

checks if the given string is a `data-*` attribute.

```ts
function (attr: string): boolean

```

- `attr` : html attribute.

### `keyFromDataAttr()`

get key from a `data-*` attribute.

- `"data-title"` => `"title"`

- `"my-attr"` => `undefined`

```ts
function (attr: string): string | undefined

```

- `attr` : html attribute.

### `getDomPropertyByAttribute()`

retrieve the dom property value by its html attribute counterpart.

```ts
function <T = string>(attr: string, el: Element): T | undefined

```

- `attr` : html attribute.
- `el`: source element.

### `isNativeToggleableAttribute()`

checks if the given attribute is natively toggleable.

```ts
function (attr: string): boolean

```

- `attr` : html attribute.

### `isToggledOn()`

checks if the given attribute is toggleable and toggled on.

```ts
function (attr: string, el: Element): boolean

```

- `attr` : html attribute.
- `el`: source element.

### `toggleAttribute()`

toggle the given attribute regardless of its type.

```ts
function (attr: string, el: Element, value?: boolean): void

```

- `attr` : html attribute.
- `el`: source element.
- `value`: optional boolean.

### `setAttribute()`

set the value of an element's attribute with the given name.

```ts
function (attr: string, value: Arrayable<DomAttribute>, el: Element): void

```

- `attr` : html attribute.
- `value`: value.
- `el`: source element.

### `removeAttribute()`

remove the element's attribute with the given name.

```ts
function (attr: string, el: Element): void

```

- `attr` : html attribute.
- `el`: source element.

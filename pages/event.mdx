# Events

Handle elemen's events.

---

### `addEventListener`

Attach an event listener to an element.

```ts
function addEventListener = <T = Event, E extends Node = Element>(
  name: EventListenerName,
  callback: DomEventHandler<E, T>,
  element: E | Window,
  isCustom?: boolean
) =>  Callback;
```

- `name` : event name.
- `callback` : event handler.
- `element` : target element.
- `isCustom` : define if the given event is custom to skip name validation.

See also: [DomEventHandler](types/#domeventhandler)

---

### `isOnEventName`

Check if the given name is valid event name.

> `onclick` is valid.

> `onClick` is valid.

> `click` is not valid.

```ts
function isOnEventName(name: string): boolean;
```

- `name` : name of the event.

---

### `setEvent`

<span>`@deprecated`</span>

Add an event with the given name to the target element.

Behind the scene, we use `element.addEventListener` to add event, we store the callback in the element in a special key, so we can remove it later.

```ts
function setEvent<T = Event, E = Element>(
  name: string,
  callback: DomEventHandler<E, T>,
  element: E
): void;
```

- `name` : event's name.
- `callback` : callback.
- `element` : target element.

See also: [DomEventHandler](types/#domeventhandler)

---

### `removeEvent`

<span>`@deprecated`</span>

Removes given element named event.

```ts
function removeEvent(name: string, element: Element): void;
```

- `name` : event's name.
- `element` : target element.

See also: [DomEventHandler](types/#domeventhandler)
